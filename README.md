# Vereda

**para escritores brasileiros.**

---

Vereda é um editor de texto que some quando você começa a escrever.

Sem barra de ferramentas pedindo atenção. Sem formatação em tempo real interrompendo o pensamento. Sem nuvem obrigatória levando seu texto pra algum servidor que você não conhece.

Você abre. Escreve. A interface quase não existe.

---

## O que ele faz

**Gêneros como ponto de partida**
Antes de começar, você escolhe: Romance, Conto, Crônica, Autoficção, Roteiro, Newsletter. O editor coloca uma voz no lugar do cursor — não um tutorial, mas uma provocação. Para Romance: *"Comece pelo momento em que tudo muda. Não pela infância. Não pela cidade."* Quando você começa a digitar, o fantasma desaparece.

**Modo Foco**
Esconde a barra lateral. Só o texto.

**Modo Leitor**
Transforma o que você escreveu numa coluna editorial limpa. Como jornal impresso. Para ler o que você escreveu como leitor, não como autor.

**Verificador gramatical de português**
Dez classes morfológicas. Feito pra português brasileiro de verdade.

**Dicionário flutuante offline**
Funciona sem internet. Nenhum dado vai a lugar nenhum.

---

## PoHW — Proof of Human Work

Essa é a ideia central do projeto.

Uma assinatura matemática que verifica que aquele texto foi *digitado* — não gerado, não colado de um modelo de linguagem. Um registro do processo humano de escrever: ritmo de digitação, pausa entre palavras, o tempo real que aquele texto levou pra existir.

Nos próximos anos, a pergunta mais importante da publicação literária vai ser: **como provar que um texto é humano?**

O Vereda tenta responder isso desde a primeira linha.
